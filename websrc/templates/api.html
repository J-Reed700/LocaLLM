API Documentation

Text Generation

Generate text responses using your local LLM.

POST /htmx/generate/text/
Content-Type: application/json

{
  "prompt": "Your text prompt here",
  "max_length": 1000,
  "temperature": 0.7,
  "conversation_id": 1,
  "type": "text",
  "name": "model-name"
}

Image Generation

Generate images using your local image model.

POST /htmx/generate/image/
Content-Type: application/json

{
  "prompt": "Your image prompt here",
  "resolution": "512x512",
  "type": "image",
  "name": "model-name"
}

Conversations

Manage chat conversations and messages.

POST /conversations/
Content-Type: application/json

{
  "title": "Conversation Title",
  "type": "text",
  "name": "model_name"
}

POST /conversations/{conversation_id}/messages
Content-Type: application/json

{
  "role": "user",
  "content": "Message content",
  "metadata": {}
}
